<h3>Introduction</h3>
<p>
  Server for the simulation of the Cover-Your-Assets card game. 
  One user creates an instance of the game, the game creates a unique 4-letter code which other users use to join the game.
  When enough users have joined, the game creator starts the game, each user is redirected to an interface which
  provide instance feedback of what the player in turn is doing. The game is played in a round until the deck is exhausted and
  no player has no card in hand.
</p>
<h3>Language</h3>
<p>Java 1.8</p>
